# 03-entrega-final

Entrega final del proyecto del módulo MP6 del Máster en IA y Big Data.

Este repositorio contiene la memoria final, notebooks, código fuente y documentación relacionados con el proyecto de predicción de ventas y segmentación de clientes en e-commerce utilizando técnicas de regresión, series temporales y clustering sobre el dataset E-Commerce Data (Kaggle/UCI). :contentReference[oaicite:0]{index=0}

El proyecto incluye:
- Predicción de ventas diarias mediante modelos como Prophet, ARIMA, Random Forest, XGBoost y LSTM.
- Segmentación de clientes mediante K-Means, Clustering Jerárquico, DBSCAN y HDBSCAN.
- Procesos de limpieza, detección de outliers, feature engineering y análisis visual de resultados. :contentReference[oaicite:1]{index=1}
